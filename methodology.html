Research Approach & Implementation

Research Design

The project employs a mixed-method approach combining qualitative and quantitative analysis. Key research methods include literature review, data collection, and pilot project implementation.

Data Collection Methods

  • Institutional Records: Gathering data on waste generation from institutions and communities.
  • Surveys & Interviews: Engaging stakeholders in waste management to gain insights.
  • Sensor Deployment: Using sensors for real-time data on waste quantities and processing efficiency.

Pilot Project Implementation

The pilot project includes setting up a biogas production system and a glass recycling unit. The implementation process consists of:

  • Glass Recycling: Collection, crushing, and conversion of glass waste into fine sand.
  • Biogas Production: Anaerobic digestion of food waste to generate renewable energy.
  • Software Development: Creating a waste monitoring system to optimize waste management strategies.
